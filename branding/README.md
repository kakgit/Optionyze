# Optionyze brand assets

These files reuse the approved Optionyze shield + OZ mark so the same image can be used consistently across mobile, web, and browser surfaces.

## Primary source

- `optionyze-app-icon-1254.png` — original master copy used for Android packaging

## Shared assets

- `shared/optionyze-logo-master.png` — general-purpose master logo
- `shared/optionyze-logo-square-512.png` — preferred square logo for app/web use
- `shared/optionyze-logo-square-256.png` — lighter square logo for UI placements
- `shared/favicon-32.png` — browser tab icon
- `shared/favicon-16.png` — small browser tab icon fallback
- `shared/apple-touch-icon.png` — iPhone/iPad home-screen icon
- `shared/android-chrome-192.png` — PWA / web app icon
- `shared/android-chrome-512.png` — large PWA / installable web app icon

## Android app use

- Launcher icon files live in `app/src/main/res/mipmap-*`
- In-app logo currently reuses `app/src/main/res/drawable-nodpi/ic_launcher_source.png`

## Web app use

If the main Optionyze web app is in a separate repo/project, copy these into that app's public/static asset folder:

- `favicon-16.png`
- `favicon-32.png`
- `apple-touch-icon.png`
- `android-chrome-192.png`
- `android-chrome-512.png`
- `optionyze-logo-square-512.png`
